# 安全指南

## 必须的安全检查

在任何提交前：
- [ ] 无硬编码密钥（API 密钥、密码、令牌）
- [ ] 所有用户输入已验证
- [ ] SQL 注入防护（参数化查询）
- [ ] XSS 防护（清理 HTML）
- [ ] CSRF 保护已启用
- [ ] 认证/授权已验证
- [ ] 所有端点有速率限制
- [ ] 错误消息不泄露敏感数据

## 密钥管理

- 绝不在源代码中硬编码密钥
- 始终使用环境变量或密钥管理器
- 启动时验证所需密钥是否存在
- 轮换任何可能已暴露的密钥

## 安全响应协议

如果发现安全问题：
1. 立即停止
2. 使用 **security-reviewer** agent
3. 在继续前修复 CRITICAL 问题
4. 轮换任何已暴露的密钥
5. 审查整个代码库是否存在类似问题
