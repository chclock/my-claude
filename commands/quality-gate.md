# 质量门命令

按需对文件或项目范围运行 ECC 质量管道。

## 使用方法

`/quality-gate [path|.] [--fix] [--strict]`

- 默认目标：当前目录（`.`）
- `--fix`：在配置的地方允许自动格式化/修复
- `--strict`：在支持的地方对警告失败

## 管道

1. 检测目标的语言/工具链。
2. 运行格式化程序检查。
3. 在可用时运行 lint/类型检查。
4. 生成简短的修复列表。

## 说明

此命令反映 hook 行为，但由操作员调用。

## 参数

$ARGUMENTS:
- `[path|.]` 可选目标路径
- `--fix` 可选
- `--strict` 可选
